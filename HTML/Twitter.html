Packet classification for mobile applications¶

We have developed the project related to the classification of encrypted packets. In the following we are going to find an algorithm that classifies different actions of a given application. The data structures that we have chosen to evaluate the results are a random forest and a neural network.

In the following box we read the dataset from a file, then we filter it given a specific application. Firstly, we have to set the path of the dataset, and the application that you want to investigate. Possible choice of application are:

#Weight
w_pos = 0.67
w_neg = 0.33

def media(array):
    a = ast.literal_eval(array)  
    sum = 0
    w_sum = 0
    for i in a:
        i=float(i)
        if(i>=0):
            sum+=w_pos*i
            w_sum+=w_pos           
        else:
            i=-i
            sum+=w_neg*(i)
            w_sum+=w_neg       
    return (sum)/(w_sum)


def variance(array, media):
    array = ast.literal_eval(array)
    media = float(media)
    sum = 0
    w_sum = 0
    for i in array:
        i=float(i)
        if(i>=0):
            sum+=  ((i-media)*(i-media))*w_pos
            w_sum+=w_pos            
        else:
            i = -i
            sum += ((i-media)*(i-media))*w_neg
            w_sum+=w_neg            
    var2 = (sum)/(w_sum) 
    var_rad = np.sqrt(var2)
    return var_rad

In the following box we calculate all the other features defined in the first box. First of all, we set a limit time. If the time distance between two different flows is more than a threshold, we consider them as two different blocks. We use the same value provided in the paper since it has been proved experimentally that 95% of all packets arrive at most 4.43 seconds after their predecessors. Then we iterate over the dataset to calculate the following features:

Num_flow = number of flow in a block¶

Position = the index of the flow in a block¶

Duration = Duration in time of the block¶

Shift_from_start = difference in time between the current flow and the first flow in a block¶

Avg_flow_length = (max time in the block - min time in block)/duration¶

Mean_block = mean combining all the mean of the flow in a block¶

Var_block = variance combining all the variance of the flow in a block¶

#This function prints and plots the confusion matrix.
def plot_confusion_matrix(cm, classes,
                          normalize=False,
                          title='Confusion matrix',
                          cmap=plt.cm.Blues):
    
    plt.figure(figsize=(12,12))
    
    plt.imshow(cm, interpolation='nearest', cmap=cmap)
    plt.title(title)
    plt
    plt.colorbar()
    tick_marks = np.arange(len(classes))
    plt.xticks(tick_marks, classes, rotation=45)
    plt.yticks(tick_marks, classes)

    if normalize:
        cm = cm.astype('float') / cm.sum(axis=1)[:, np.newaxis]

    thresh = cm.max() / 2.
    for i, j in itertools.product(range(cm.shape[0]), range(cm.shape[1])):
        plt.text(j, i, cm[i, j],
                 horizontalalignment="center",
                 color="white" if cm[i, j] > thresh else "black")

    plt.tight_layout()
    plt.ylabel('True label')
    plt.xlabel('Predicted label')

from sklearn.utils import shuffle 
from keras import models
from keras import layers

def NN_model():
    model = models.Sequential()
    model.add(layers.Dense(512, activation = "relu", input_shape=(selected_features.shape[1],)))
    
    # Hidden - Layers    
    model.add(layers.Dense(256, activation = "relu"))
    model.add(layers.Dropout(0.3, noise_shape=None, seed=None))
    model.add(layers.Dense(128, activation = "relu"))
    model.add(layers.Dropout(0.3, noise_shape=None, seed=None))
    model.add(layers.Dense(64, activation = "relu"))
    model.add(layers.Dropout(0.3, noise_shape=None, seed=None))
    model.add(layers.Dense(32, activation = "relu"))
    model.add(layers.Dropout(0.3, noise_shape=None, seed=None))
    
    # Output- Layer
    model.add(layers.Dense(num_classes, activation = "softmax"))
    
    model.compile(loss='categorical_crossentropy', optimizer='adam', metrics=['accuracy'])
    return model
